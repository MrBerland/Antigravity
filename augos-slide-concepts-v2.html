// v2 — BOLDER CONCEPTS

5 Fundamentally Different Slide Systems

Each concept has a different grid logic, spatial language, and typographic system — so a 20-slide deck doesn't blur into sameness. Three slide variants are shown per concept to demonstrate how the system sustains variety.

What changed vs v1: v1 concepts shared the same spatial structure (text left, visual right) and the same dark navy palette. Here, each system has a completely different layout engine — Hard Split, Data-First, Brutalist, Editorial Magazine, and Cinematic. They don't feel like variations of each other — they're different philosophies.
// CONCEPT 01
Hard Split
Every slide is a strict 50/50 vertical divide — left half is a single massive statement in colour, right half is the supporting argument on white. The colour of the left half changes per slide to signal context: black=authority, red=problem, green=result, blue=data. This colour logic is the memory system.
SYSTEM: colour left ← context · right always white · number always bleeds off left edge

Editorial Magazine
Warm off-white paper + Augos navy. Three completely different column grid layouts that cycle through the deck — wide headline + narrow columns, pull quote dominant, timeline narrative. Inspired by Bloomberg and Swiss International Style. Variety is structural, not decorative.
