Charting

The most flexible tool in the platform. Build ad-hoc custom parameter visualisations by selecting measurement points and parameters, then render them as independent Highcharts panels with per-chart resolution controls.

1. Capability & Purpose

While other reports serve predefined analytical views, Charting gives energy managers the freedom to investigate any hypothesis. Select a measurement point, choose parameters (consumption, demand, power factor, voltage), and generate custom chart panels — each with independent resolution controls.

  • Parameter Selection Wizard: Multi-step flow — Search point → Select parameters (checkbox) → Generate.
  • Dynamic Chart Panels: One Highcharts panel per selected parameter. Each with independent Min / Halfhour / Day / Month resolution tabs.
  • Edit/Delete Per Panel: Add or remove charts without page reload — iterate quickly between hypotheses.
  • Highcharts Engine: Unlike other reports that use Recharts, Charting uses Highcharts for maximum interactivity (zoom, tooltip, export).
