Load Curtailment

The only report that defaults to "today". Real-time load shedding compliance with 1-minute resolution, CBL target tracking, 6-tab performance analysis, and a 12-month compliance calendar — the mission-critical tool during active Eskom load shedding events.

1. Capability & Purpose

During Eskom load shedding events, large power users must curtail to their CBL (Curtailment Base Load). Non-compliance can result in disconnection or financial penalties. Load Curtailment provides real-time tracking against the target with the highest temporal resolution in the platform.

  • 1-Minute Resolution: The finest granularity in the platform — essential for tracking compliance during active curtailment events.
  • Stage Status Header: Real-time Eskom load shedding stage (Stage 0–8) displayed prominently.
  • 4 KPI Cards: Current Trajectory (kW), Over Achieved (kWh), Site Compliant (✅/❌), Node Compliant (✅/❌).
  • 6-Tab Performance Section: Site Performance, Node Performance, Site CBL Calculation, Site Contribution to Node Target, Trade Capacity, Trading Dashboard.
  • 12-Month Compliance Calendar: Load shedding events (red), abnormal exclusions (green), public holidays (amber).
Real-Time Compliance

During an active Stage 4 load shedding event, the operations manager watches the Load Curtailment report in real-time. The Actual Load line must stay below the CBL Target line. If it crosses, the "Site Compliant" card flips to ❌ red. The 1-minute resolution means they can react within seconds — shutting down non-essential equipment to avoid compliance penalties.
