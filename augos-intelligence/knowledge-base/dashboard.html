Dashboard

The single-screen operational health check for energy management. The Dashboard aggregates site KPIs, consumption heatmaps, demand/power charting, and full meter hierarchy status into one view — designed to be scanned in under 10 seconds at 08:00 every morning.

1. Capability & Purpose

The Dashboard is the entry point for every Augos Energy Intelligence user. It answers the five morning-check questions a facility manager needs before the day even starts:

  • Is data flowing? Last Activity timestamps confirm every meter is reporting. A stale timestamp signals a comms failure.
  • Was yesterday normal? The consumption heatmap shows 6 months of colour-coded daily patterns. Red cells demand investigation.
  • What's our demand? Peak demand peaked at 803 kVA — is that within budget?
  • Is power factor healthy? A PF below 0.90 flashes immediate concern.
  • Are there seasonal patterns? The heatmap's 6-month view reveals trends invisible in daily data.
The 10-Second Morning Check

A facility manager checks the Dashboard at 08:00 every morning. In 10 seconds they confirm: all meters are reporting (Last Activity timestamps are fresh), yesterday's consumption was normal (heatmap shows no red cells), demand peaked at 803 kVA (within budget), and power factor is 0.99 (healthy). If anything is wrong, they see it immediately — a stale timestamp, a red heatmap cell, a PF below 0.90 — and they investigate before it becomes a crisis.

2. Key Components

Component Function Interaction
4 Summary Cards KPI metrics with embedded stacked bar mini-charts Click to navigate to detailed report
Consumption Heatmap 6-month bubble chart (Mon–Sun × Week), colour-coded green→red Toggle: Consumption / Demand-Power / Power Factor
Technical Analysis Table Point, Last Activity, Consumption (kWh), Demand (kVA), PF Show Statistics toggle expands per-phase data
Demand/Power Chart Highcharts time-series: Demand (blue), Power (purple) 2-year range slider for trend exploration
Full Meter Hierarchy Expandable concertina: site → main switches → sub-meters Click to expand/collapse tree branches

Fuel Type Support

The Dashboard supports four fuel types via tabs: Electricity, Water, Fuel, and Gas. Each renders the same component structure but with fuel-specific parameters and units (kWh, kL, litres, m³).

Morning health check Check if invoice arrived. No real-time view. 10-second scan: all meters live, consumption normal, PF healthy.

5. Data Architecture & Integration

The Dashboard API synthesizes data from multiple underlying services into a single payload. This enables the AI-Ready file export to contain a holistic site snapshot.

{
  "endpoint": "/api/v1/dashboard",
  "method": "GET",
  "parameters": {
    "pointId": 8323,
    "startDateUTC": "2026-02-24T22:00:00.000Z",
    "endDateUTC": "2026-02-25T21:59:59.998Z",
    "productTypeId": 1
  },
  "response": {
    "summaryCards": [ /* 4 KPI objects */ ],
    "heatmapData": [ /* 6-month daily consumption array */ ],
    "meterHierarchy": {
      "root": "Cape Town (8323)",
      "children": [
        { "name": "Main Switch 1", "pointId": 8324, "kWh": 77127 },
        { "name": "Main Switch 2", "pointId": 8336, "kWh": 60391 }
      ]
    },
    "demandPowerChart": { /* Highcharts series data */ }
  }
}
