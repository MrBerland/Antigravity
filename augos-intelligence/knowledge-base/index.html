AI-Ready Data Architecture

Every analytical report generates a downloadable AI-Ready file — structured for LLM consumption with embedded guidelines.

File Structure

  • Format.min.json (minified JSON)
  • Size Range30–454 KB per report
  • Available On20 of 22 reports
  • Top-Level Keys_aiInstructions + data
  • Guidelines9 embedded LLM consumption rules
  • Download FlowBanner → Disclaimer → Checkbox → Download

Embedded LLM Guidelines

  1. Base responses strictly on the data provided
  2. Avoid assumptions or external sources
  3. No unsupported commentary
  4. State clearly when information is missing
  5. Reference specific values or time periods
  6. Keep explanations clear and jargon-free
  7. Use bullet points or numbered lists
  8. Highlight anomalies only if data supports them
  9. Maintain neutral tone
